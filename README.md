# 3D Zero-Gravity Pool - Cosmic Nebula Edition 🌌

A fully immersive, sci-fi themed **3D Zero-Gravity Billiards / Pool simulation** built using **Three.js** and rendered seamlessly inside a **Streamlit** web application via custom HTML/JS component integration.

Navigate deep-space orbital coordinates, command holographic laser cues, throttle plasma energy levels, and sink the Obsidian 8-Ball into a corner gravitational singularity without losing your cue ball to cosmic space-drift!

\---

## ✨ Features

* **Immersive 3D Physics \& Graphics**: Powered by Three.js with custom particle nebula backgrounds, reflection mapping, and dynamic lighting.
* **Sci-Fi Control Console**:

  * **Azimuth Aim Radar Joystick**: Smooth touch and mouse-compatible directional aiming joystick with animated radar sweeps.
  * **Engine Throttle Lever**: Adjust your launch strike power dynamically using an interactive space throttle track.
  * **Audio Synthesis Engine**: Real-time Web Audio API sound effects for laser pulls, cushion thuds, ball-to-ball clacks, and spatial pocket warps.
* **Responsive HUD Overlays**: Integrated welcome briefings, warning hazard alerts for gravity well scratches, victory orbit celebrations, and mission abort menus.

Make sure you have Python installed, then install Streamlit:
